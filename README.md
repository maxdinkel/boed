# Bayesian Optimal Experimental Design on Finite Design Sets

This repository contains an implementation of the methods presented in the paper *"Efficient Bayesian Optimal 
Experimental Design for Expensive Computational Models over Finite Design Sets"* together with scripts for reproducing 
the experiments reported in the paper.

## Installation

```bash
git clone https://github.com/maxdinkel/boed.git
cd boed
conda env create -f environment.yml
conda activate boed
pip install .
```

## Reproducing the Paper Experiments

The scripts used to reproduce the experiments from the paper are provided in the paper/ directory. 
See paper/README.md for instructions.
